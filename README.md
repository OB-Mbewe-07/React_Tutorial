# 🎬 Anime Showtimes App

A modern React + TypeScript application that displays anime listings with available showtimes in a clean, responsive UI.

This project demonstrates component-based architecture, type-safe props, dynamic rendering with `.map()`, and modern CSS styling techniques.

## ✨ Features

- ⚛️ Built with React
- 🟦 Fully typed using TypeScript
- 🧩 Reusable and modular components
- 🕒 Dynamic showtime rendering
- 🪟 Modal with controlled event propagation
- 🎨 Custom styled UI
- 📱 Responsive layout
